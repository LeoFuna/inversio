# Bem vindo ao Sistema Inversio!

Olá seja bem vindo(a) ao meu projeto Inversio, o projeto é um sistema web para controle e tratamento de dados referentes à operações de day trade no mercado brasileiro.

A idéia desse projeto veio de uma demanda própria, onde as plataformas de trades atuais tem um tratamento de dados muito superficial, e eu sentia falta de poder ter um tratamento melhor, algo que me mostrasse em detalhes onde eu vou bem e onde vou mal, para assim poder ir melhorando.

Como se trata de uma demanda, onde acredito que outros possam ter, eu pretendo que esse sistema possa ser comercializado em um modelo Freemium.
